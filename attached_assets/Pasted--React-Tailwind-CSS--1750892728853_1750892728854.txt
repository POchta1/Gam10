Создай современный веб-сайт для компании "Школа английского" с использованием React и Tailwind CSS. Сайт должен быть оформлен в бело-красной цветовой гамме (или близких оттенках, например, мягкий белый и яркий красный). Дизайн должен быть эстетичным, профессиональным и современным, с акцентом на красоту и интерактивность.

Основные требования:





Дизайн и анимации:





Используй плавные, продуманные анимации, адаптированные под действия пользователя (например, эффекты при наведении, прокрутке, загрузке страниц, переходах между секциями).



Анимации должны быть ненавязчивыми, но подчеркивать современность сайта (например, fade-in, scale, parallax при скролле).



Дизайн должен быть чистым, минималистичным, но с яркими акцентами красного цвета для привлечения внимания.



Структура сайта:





Сайт должен быть большим, с большим количеством информации о школе. Включить следующие разделы:





Главная страница: Приветственный баннер с эффектной анимацией, краткое описание школы, призыв к действию (например, "Запишитесь на пробный урок").



О школе: История, миссия, ценности, преимущества обучения, информация о преподавателях (с фотографиями и краткими профилями).



Программы обучения: Подробное описание курсов (для детей, взрослых, бизнес-английский и т.д.) с ценами и возможностью выбора уровня.



Отзывы: Карусель или сетка с отзывами учеников, с анимацией появления.



Контакты: Карта (например, Google Maps), форма обратной связи, адрес, телефон, email.



Форма расценок: Интерактивная форма, где пользователь может выбрать курс, уровень и получить предварительную стоимость. Форма должна быть анимированной и удобной.



Блог/Новости: Секция с полезными статьями или новостями школы (3–5 примеров статей с анимацией карточек).



FAQ: Раздел с часто задаваемыми вопросами и ответами в формате аккордеона.



Технические требования:





Используй React с JSX и Tailwind CSS для стилизации.



Подключи зависимости через CDN (например, с cdn.jsdelivr.net для React, ReactDOM, Babel).



Сайт должен быть одностраничным (SPA) с плавной навигацией и анимацией переходов между секциями.



Обеспечь адаптивность для всех устройств (мобильные, планшеты, десктопы).



Сайт должен запускаться просто через npm start с использованием Vite или Create React App. Никаких сложных конфигураций сервера или дополнительных команд.



Код должен быть чистым, модульным, с переиспользуемыми компонентами React.



Язык и локализация:





Сайт полностью на русском языке.



Текст должен быть профессиональным, дружелюбным и мотивирующим к обучению.



Дополнительно:





Добавь интерактивные элементы, такие как кнопки с ховер-эффектами, анимированные иконки, прогресс-бары для демонстрации успехов учеников.



Используй современные шрифты (например, Google Fonts: Roboto, Montserrat или аналогичные).



Оптимизируй производительность сайта (ленивая загрузка изображений, оптимизация анимаций).



Убедись, что сайт выглядит премиально и отражает высокое качество услуг школы.

Примерный стек технологий:





React (с JSX)



Tailwind CSS (через CDN или npm)



Vite или Create React App для сборки



React Router для навигации



Framer Motion или GSAP для анимаций



Google Fonts для шрифтов



CDN для всех внешних зависимостей

Результат:

Сгенерируй полный код для сайта (index.html, App.jsx, необходимые компоненты и стили), который можно запустить одной командой npm start. Код должен быть готов к развертыванию, с акцентом на красоту, современность и интерактивность. Включите примеры текстов для всех разделов на русском языке, чтобы сайт выглядел заполненным и готовым к использованию.
